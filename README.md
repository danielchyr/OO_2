2. 文字敘述專題需求：
>考量到出門外食，夜市、餐館等人多的字方用餐。若有方便的點餐系統來協助消費者可以使用自己的載具點餐，商家既可以省下人力成本、又用平板系統管理訂單，降低錯誤的機率。
3. 寫出專題之功能性需求與非功能性需求：
>功能性
* 點餐:使用自己的載具，簡化點餐手續。
* 查詢:使用者和店家都可以查詢訂單詳細資料。
>非功能性
* 效能:系統能隨時更新資料庫，並將訂單和取餐訊息即時推撥給店家和消費者。
